# アカウント権限

                
        
                > 💡 ✅POPstationでは、アカウントごとに1つの権限を設定します。権限に応じて利用できる機能が異なります。
                > 💡 目次
                
        
                
        ### 権限一覧
        
                ---
        
    | 権限 | チャット利用 | モード管理 | カスタムデータ管理 | カスタムデータグループ管理 | ユーザーグループ管理 | ユーザー管理 | システム管理 | IPアドレス制限なし |
    | --- | --- | --- | --- | --- | --- | --- | --- | --- |
    | 管理者 | 〇 | ○
 | 〇 | 〇 | 〇 | 〇 | 〇 | 〇 |
    | ユーザー管理者 | 〇 | ○ | 〇 | 〇 | 〇 | 〇 | 〇 |  |
    | カスタムデータ管理者 | 〇 | △ ※1 | 〇 |  |  |  |  |  |
    | 一般ユーザー | 〇 | △ ※1 |  |  |  |  |  |  |

                ※1 自身が作成したモードのみ編集可能
        
                
        #### 各権限の説明
        
                * 管理者: システム全体に関する全ての操作が可能です。ユーザー管理、システム設定、カスタムデータ管理など、POPstationの全ての機能を制限なく利用できます。
                * ユーザー管理者: ユーザー管理とシステム設定に関する操作が可能です。ユーザーの追加、削除、アカウント情報編集、権限設定などを行い、システムの運用に必要な設定変更を行うことができます。
                * カスタムデータ管理者: カスタムデータのアップロード、削除、編集など、カスタムデータに関する全ての操作が可能です。自社データ利活用機能を利用する環境でのみ設定可能です。
                * 一般ユーザー: POPstationの基本的な機能であるチャットとカスタムデータの利用が可能です。ユーザー管理やシステム設定など、システムに関する操作はできません。
                
        
                
        #### 各機能の説明
        
                * チャット利用: POPstationのチャット機能を利用できます。
                * モード管理: モードの作成、編集、削除など、モードに関する操作が可能です。
                * カスタムデータ管理: カスタムデータのアップロード、削除、編集など、カスタムデータに関する操作が可能です。
                * カスタムデータグループ管理・ユーザーグループ管理: カスタムデータやAIモデルをグループ単位で制御するための、カスタムデータグループとユーザーグループの作成、編集、削除が可能です。
                * ユーザー管理: ユーザーの追加、削除、アカウント情報編集、権限設定など、ユーザーに関する操作が可能です。
                * システム管理: システム設定の変更など、システムに関する設定が可能です。
                * IPアドレス制限なし: アクセスを特定のIPアドレスに制限する設定がされている場合でも、この権限を持つユーザーは制限なくアクセスできます。管理者がIPアドレス設定を誤って設定した場合でも、この権限を持つユーザーはシステムにアクセスできるよう設計されています。
