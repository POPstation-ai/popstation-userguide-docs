# ユーザー管理





### ユーザー管理画面への遷移

管理画面左側のユーザー管理をクリックすると、ユーザー管理画面が表示されます。



![画像資料](images/1e125c9e-c4d6-80e3-8d69-e5d3efd7a29b.png)


### ユーザーの追加・更新

新規ユーザーの追加を行う際、そのユーザーの権限を設定します。

権限の詳細については、[「アカウント権限」ページ](/1cb25c9ec4d681cc8d0adfeecfded9de)をご確認ください。



新規ユーザー追加・更新の方法には以下の2通りがあります。

* ユーザー毎に追加
* Excelファイルを使用して複数のユーザーを一括で追加・更新
追加・更新するユーザーが多い場合は、Excelファイルを使用することを推奨します。

#### Excelファイルを使用した一括追加・更新

ユーザーの一括追加・更新を行いたい場合は、ユーザー追加用のCSVファイルをアップロードすることで行うことができます。ユーザー追加だけでなく、ユーザー情報を上書き更新したい場合も可能です。



![画像資料](images/1e125c9e-c4d6-80ef-bc74-ee87788f4dc7.png)


![画像資料](images/1cb25c9e-c4d6-8154-8507-fbbd575b0042.png)


![画像資料](images/1cb25c9e-c4d6-8105-9c2c-c3204f6265c7.png)
* 各権限は以下のように記述することで定義できます。
* ユーザーグループの詳細については、[「グループ管理」ページ](/1cb25c9ec4d6815d9c30f12e8298464f)をご覧ください。
