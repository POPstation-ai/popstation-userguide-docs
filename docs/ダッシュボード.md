# ダッシュボード



### ダッシュボードの確認



![画像資料](images/1dd25c9e-c4d6-804b-8919-e284f17da2d6.png)


![画像資料](images/1dd25c9e-c4d6-80e6-8391-cbecf6378056.png)


ダッシュボードでは、以下の項目を確認できます。

* ユーザ数
* 月毎の利用ユーザー数（MAU）
* 日毎の利用ユーザー数（DAU）
* 日別送信数
* 月間送信数


### 利用料金アラートの表示

ダッシュボード画面にアクセスした際に、今月のご利用金額が利用上限金額の90%以上に達している場合、アラート画面が表示されます。
